A privacy-first data governance agent for humanitarian NGOs handling beneficiary data across federated systems. Uses Auth0 Token Vault for secure credential storage, CIBA (Client-Initiated Backchannel Authentication) for human-in-the-loop approval flows on sensitive data access, and IBM Granite 4 for on-prem reasoning over case records. The name (Arabic: أمانة) means a sacred trust — something held in stewardship for someone else.
